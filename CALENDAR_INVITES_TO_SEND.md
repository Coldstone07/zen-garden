## HOW TO SEND THESE

**Option 1: Google Calendar**
1. Create each event in Google Calendar
2. Invite attendees by email address
3. Paste the description text into the event notes
4. Send invite

**Option 2: Outlook**
1. Create event in Outlook
2. Add attendees
3. Paste description into notes/body
4. Send invite

**Option 3: Email (if calendar integration not available)**
1. Email the attendees with subject line: "[CALENDAR INVITE] [Event Title]"
2. Include all details (date, time, link, agenda) in email body
3. Confirm attendance via reply
